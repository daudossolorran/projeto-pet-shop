# Projeto Pet Shop

## Descrição

Este projeto consiste em um **site institucional responsivo para uma pet shop**, desenvolvido utilizando **HTML** e **CSS**. O objetivo é criar uma presença online atraente e funcional para o negócio, apresentando seus serviços, produtos e informações de contato de forma clara e acessível em diferentes dispositivos. O foco principal é a usabilidade e um design que transmita carinho e profissionalismo.

## Tecnologias Utilizadas

- **HTML**: Estrutura semântica do conteúdo da página.
- **CSS**: Estilização visual, layout e responsividade para diversos tamanhos de tela.

## Funcionalidades

- Página inicial com destaque para os serviços.
- Seção de produtos com informações detalhadas.
- Formulário de contato para agendamentos e dúvidas.
- Design responsivo para desktop, tablet e mobile.

## Como Visualizar

1.  **Acesse a aplicação**: [Link para a demo ao vivo (se disponível)](https://daudossolorran.github.io/projeto-pet-shop/ )
2.  **Navegue pelas seções**: Explore os serviços, produtos e a página de contato.

## Instalação Local (para desenvolvedores)

Para rodar o projeto localmente, siga os passos abaixo:

1.  **Clone o repositório**:
    ```bash
    git clone https://github.com/daudossolorran/projeto-pet-shop.git
    ```
2.  **Navegue até o diretório do projeto**:
    ```bash
    cd projeto-pet-shop
    ```
3.  **Abra o arquivo `index.html`** no seu navegador preferido.
