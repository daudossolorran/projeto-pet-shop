# Pet Boutique

Este projeto é um site institucional desenvolvido com HTML5 e CSS3 para uma pet shop fictícia chamada **Pet Boutique**.

## O que foi feito

- Estruturação completa da página com HTML5 semântico.
- Estilização com CSS3 utilizando classes organizadas e responsividade básica.
- Criação de seções como:
  - Cabeçalho com logo, nome e slogan.
  - Seção de apresentação dos serviços oferecidos.
  - Seção com imagens ilustrativas dos serviços.
  - Destaque para redes sociais com links funcionais.
  - Apresentação de marcas parceiras com logos.
  - Mapa integrado com Google Maps para exibir a localização da loja.
  - Rodapé com créditos de desenvolvimento.
- Integração com WhatsApp para contato direto.
- Organização do conteúdo com containers e classes reutilizáveis.

## Tecnologias

- HTML5
- CSS3
